# Express Integration & Simplified API Plan

**Status**: Design Proposal
**Created**: 2026-02-22
**Updated**: 2026-02-22
**Target**: Add Express integration with clean, simplified API

---

## Overview

Add Express.js/http.Server integration to `@synnel/server` with a clean, simplified API.

**Key Design Principle**: Channels must be explicitly created on the server before clients can join them. No dynamic channel creation.

**Target API:**
```typescript
import express from 'express'
import { Synnel } from '@synnel/server'

const app = express()
const httpServer = app.listen(3000)

// Initialize with Express server
const synnel = new Synnel(httpServer)

// Define channels - only these channels can be joined by clients
const chat = synnel.multicast('chat')
const notifications = synnel.broadcast()

// Use channels
chat.receive((data, client) => console.log(data))
chat.send({ text: 'Welcome!' })
notifications.send({ announcement: 'Hello everyone!' })
```

---

## Goals

1. Accept `http.Server` directly in constructor
2. Create standalone HTTP server if no server provided
3. Add two convenience methods: `multicast()`, `broadcast()`
4. **Enforce explicit channel creation** - clients can only join channels created via `multicast()` or `broadcast()`
5. Export main `Synnel` class as the primary API

### Channel Creation Model

```typescript
class Synnel {
  private createdChannels: Set<ChannelName> = new Set()

  multicast<T>(name: string): ChannelTransport<T> {
    // Mark this channel as created/allowed
    this.createdChannels.add(name)
    return this.channel<T>(name)
  }

  broadcast<T>(): BroadcastTransport<T> {
    // '__broadcast__' is implicitly allowed
    return this.broadcastTransport<T>()
  }
}
```

### Subscription Validation

```typescript

      data: { message: `Channel '${channel}' is not allowed`, code: 'CHANNEL_NOT_ALLOWED' }
    })
    return
  }
  // ...proceed with subscription
}
```

## Implementation Details

### multicast()

```typescript
/**
 * Create a multicast channel (many-to-many messaging)
 * This allows clients to subscribe to this channel
 * @param name - Channel name
 * @returns Channel transport with send/receive methods
 */
multicast<T = unknown>(name: string): ChannelTransport<T> {
  // Mark this channel as created/allowed
  this.createdChannels.add(name)
  return this.channel<T>(name)
}
```

### broadcast()

```typescript
/**
 * Create a broadcast transport (server-to-all messaging)
 * @returns Broadcast transport
 */
broadcast<T = unknown>(): BroadcastTransport<T> {
  return this.broadcastTransport<T>()
}
```

### Subscription Enforcement

```typescript
private async handleSignal(
  serverClient: ServerClient,
  signal: SignalType,
  channel: ChannelName | undefined,
): Promise<void> {
  if (!channel) {
    return
  }

  if (signal === CoreSignalType.SUBSCRIBE) {
    // NEW: Check if channel was explicitly created
    if (channel !== '__broadcast__' && !this.createdChannels.has(channel)) {
      await serverClient.send({
        id: `error-${Date.now()}`,
        type: MessageType.ERROR,
        data: {
          message: `Channel '${channel}' is not allowed`,
          code: 'CHANNEL_NOT_ALLOWED',
        },
        timestamp: Date.now(),
      })
      return
    }

    // Execute subscribe middleware
    try {
      await this.middleware.executeSubscribe(serverClient, channel)
    } catch (error) {
      // ...error handling
    }

    // ...proceed with subscription
  }
}
```

```typescript
import express from 'express'
import { Synnel } from '@synnel/server'

const app = express()
const server = app.listen(3000)

const synnel = new Synnel(server)

// Create channels - ONLY these channels can be joined

const notifications = synnel.broadcast()

// Use channels
chat.receive((data, client) => console.log(data))
chat.send({ text: 'Welcome!' })
notifications.send({ announcement: 'Hello everyone!' })
```

### Standalone Server

```typescript
import { Synnel } from '@synnel/server'

const synnel = new Synnel({ port: 3000 })

// Create channels first
const chat = synnel.multicast('chat')

// Now clients can join 'chat'
```

### Channel Enforcement

```typescript
// Server
const synnel = new Synnel(server)
synnel.multicast('chat')           // ✅ Clients CAN join
synnel.multicast('notifications') // ✅ Clients CAN join
// 'admin' channel NOT created     // ❌ Clients CANNOT join

// Client trying to join 'admin'
await client.subscribe('admin')    // ❌ ERROR: Channel not allowed
```

### With Authorization

```typescript
synnel.authorize(async (clientId, channel, action) => {
  if (channel === 'chat') {
    return await isAuthenticated(clientId)
  }
  return true
})
```
